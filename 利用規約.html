利用規約

更新しました2023 年 3 月 23 日

最近、使用ポリシーをより明確かつ具体的に更新しました。


私たちは、誰もが私たちのツールを安全かつ責任を持って使用することを望んでいます。そのため、OpenAI のモデル、ツール、およびサービスのすべてのユーザーに適用される使用ポリシーを作成しました。これらに従うことで、当社のテクノロジーが確実に有効に使用されるようになります。

お客様の製品または使用法がこれらのポリシーに従っていないことが判明した場合、必要な変更をお願いする場合があります。繰り返しまたは重大な違反があった場合、アカウントの一時停止または終了を含むさらなる措置が取られる可能性があります。

モデルの使用と悪用についてさらに理解が深まるにつれて、ポリシーが変更される可能性があります。

モデルの使用禁止

次の目的でモデルを使用することは許可されていません。

  • 違法行為
    • OpenAI は、当社のモデル、ツール、およびサービスを違法行為に使用することを禁止しています。
  • 児童の性的虐待の素材、または児童を搾取または害するコンテンツ
    • 私たちは、行方不明および搾取された子供のための国立センターに CSAM を報告します。
  • 憎悪、嫌がらせ、または暴力的なコンテンツの生成
    • アイデンティティに基づいて憎悪を表現、扇動、助長するコンテンツ
    • 個人への嫌がらせ、脅迫、またはいじめを目的とするコンテンツ
    • 暴力を助長または美化するコンテンツ、または他人の苦しみや屈辱を称賛するコンテンツ
  • マルウェアの生成
    • コンピュータ システムを妨害、損傷、または不正アクセスするように設計されたコードを生成しようとするコンテンツ。
  • 以下を含む、身体的危害のリスクが高い活動:
    • 兵器開発
    • 軍事と戦争
    • エネルギー、輸送、および水における重要なインフラストラクチャの管理または運用
    • 自殺、切断、摂食障害などの自傷行為を助長、助長、または描写するコンテンツ
  • 以下を含む、経済的損害のリスクが高い活動:
    • 連鎖販売取引
    • ギャンブル
    • ペイデイローン
    • 信用、雇用、教育機関、または公的支援サービスの適格性の自動決定
  • 以下を含む詐欺行為または欺瞞行為:
    • 詐欺
    • 組織的な不審な行動
    • 盗作
    • 学問的不正
    • 偽の草の根サポートや偽のレビュー生成などの人工芝化
    • 偽情報
    • スパム
    • 疑似医薬品
  • 以下を含むアダルト コンテンツ、アダルト産業、出会い系アプリ:
    • 性行為の説明など、性的興奮を喚起することを意図したコンテンツ、または性的サービスを促進するコンテンツ (性教育と健康を除く)
    • エロチャット
    • ポルノ
  • 以下による政治的キャンペーンまたはロビー活動:
    • 大量のキャンペーン資料の生成
    • 特定の人口統計に合わせてパーソナライズされた、またはターゲットを絞ったキャンペーン資料の生成
    • キャンペーンに関する情報を提供したり、政治的擁護やロビー活動に関与したりするチャットボットなどの会話型または対話型のシステムを構築する
    • 政治キャンペーンやロビー活動を目的とした製品の構築
  • 以下を含む、人々のプライバシーを侵害する行為:
    • 個人の同意なしに個人を追跡または監視すること
    • 個人の顔認識
    • 保護特性に基づく個人の分類
    • 識別または評価のためのバイオメトリクスの使用
    • 個人を特定できる情報または教育、財務、またはその他の保護された記録の違法な収集または開示
  • 無許可の法律実務に従事すること、または有資格者が情報を確認することなく個別の法的助言を提供すること
    • OpenAI のモデルは、法的助言を提供するよう微調整されていません。法的助言の唯一の情報源として私たちのモデルに頼るべきではありません.
  • 有資格者が情報を確認することなく、個別の金融アドバイスを提供する
    • OpenAI のモデルは、金融アドバイスを提供するように微調整されていません。金融アドバイスの唯一の情報源として、当社のモデルに頼るべきではありません。
  • 誰かに特定の健康状態があるかどうかを伝える、または健康状態を治す方法や治療する方法を説明する
    • OpenAI のモデルは、医療情報を提供するように微調整されていません。重大な病状の診断または治療サービスを提供するためにモデルを使用しないでください。
    • OpenAI のプラットフォームは、緊急の対応が必要な生命を脅かす問題のトリアージや管理に使用しないでください。
  • 以下を含む、リスクの高い政府の意思決定:
    • 法執行と刑事司法
    • 移住と亡命

モデルの特定の用途については、さらに要件があります。

  1. 医療、金融、法律業界における当社のモデルの消費者向けの使用。ニュースの生成またはニュースの要約。また、正当な理由がある場合は、AI が使用されていることと、その潜在的な制限についてユーザーに通知する免責事項をユーザーに提供する必要があります。
  2. 自動化されたシステム (会話型 AI やチャットボットを含む) は、AI システムと対話していることをユーザーに開示する必要があります。歴史上の著名人を描いたチャットボットを除き、別の人物をシミュレートする製品は、その人物の明示的な同意を得るか、「シミュレートされた」または「パロディ」と明確にラベル付けする必要があります。
  3. ライブ ストリーム、デモンストレーション、研究でのモデル出力の使用は、 共有および公開ポリシーの対象となります。

無料のモデレート エンドポイント と 安全性のベスト プラクティスを使用して  、アプリを安全に保つことができます。

プラットフォーム ポリシー

当社の API は、多くのセクターやテクノロジー プラットフォームでビジネスを強化するために使用されています。iOS アプリから Web サイト、Slack まで、シンプルな API により、幅広いユースケースに統合できます。下記の使用例の制限に従って、すべての主要なテクノロジー プラットフォーム、アプリ ストアなどの製品に API を統合することができます。

プラグイン ポリシー

上記のモデルの禁止された使用法に加えて、 プラグインを構築する開発者には追加の要件があります。

  • プラグイン マニフェストには、モデルに公開されている API の機能と一致する明確な説明が必要です。
  • プラグイン マニフェスト、OpenAPI エンドポイントの説明、またはプラグイン応答メッセージに、無関係、不必要、または欺瞞的な用語または指示を含めないでください。これには、他のプラグインの使用を避けるための指示、またはモデルの動作を操作または設定しようとする指示が含まれます。
  • OpenAI の安全システムを回避または妨害するためにプラグインを使用しないでください。
  • 人間のような応答をシミュレートするか、事前にプログラムされたメッセージで返信するかにかかわらず、プラグインを使用して実際の人との会話を自動化しないでください。
  • ChatGPT によって生成された個人的な通信またはコンテンツ (電子メール、メッセージ、またはその他のコンテンツなど) を配布するプラグインは、コンテンツが AI によって生成されたことを示す必要があります。

他の使用ポリシーと同様に、プラグインの使用と悪用についてさらに学ぶにつれて、プラグイン ポリシーが変更されることを期待しています。
